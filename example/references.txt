The tests refer to the textbooks issued by the authors,
however I have only the 2nd edition. 
These describe an improved version FOSBIC2 which I don't have source of,
so several example code does not work.

The test*.bas files in the examples directory are working.

Between the FOSBIC1 version described here, and FOSBIC2, significant
changes in FILE handling were made, so the file*.bas examples were
created from scratch.
